Detailed Description

Codec Class - Encodes and Decodes string into an audio buffer.

The string is extracted by scanning the audio buffer for any samples where the left channel LSB is 0 and the right channel LSB is not 0, converting the LSB from ascii to char then adding as next operator in string. A Union is used to change the LSB of float sample.
